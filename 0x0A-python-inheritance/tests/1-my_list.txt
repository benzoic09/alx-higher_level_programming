=================================
The ``Print Sorted List`` module
================================

Using ``MyList``
----------------

 >>> MyList = __import__("1-my_list").MyList

Examples:

>>> a = MyList()
>>> a
[]

>>> a.append(1)
>>> a
[1]

>>> a = MyList()
>>> a.append(5)
>>> a.append(3)
>>> a
[1, 5, 3]
